Feature rich, high performance, no dependencies.
'ag' stands for Agnostic - works with any framework.

#### Features:

* Sorting
* Filtering
* Resizing
* Pinning
* Selection
* Value Getters
* Cell Styling
* Cell Rendering
* Cell Templates
* Editing Cells
* Context
* Refresh
* Grouping Headers
* Header Rendering
